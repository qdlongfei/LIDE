# LIDE
This is a lite develop framework, that support dynamic develop,dynamic deploy,dynamic monitor and can make your project deliver fastest! make your developer focus on bussiness only. to try it!

    大繁至简，现在的开发框架，不管是ssh或者是springMVC，都要依赖一大堆的库，依赖于一大堆的规则，开发、部署、维护、监控的成本都很高。当然，这种高成本是相对于项目大小来说的，当项目很大的时候，这种付出是有必要的，但是对于一般的中小项目，或者是一些临时性的项目来说就有些不值得。我们从“远古”的开发时代过来，愿意去尝试简化轮子，愿意去思考如何将复杂的轮子精简成简单的，愿意将很高大上的菜重新包装成“路边摊”，如果您也有这种想法，欢迎您加入这个项目；如果您需要这样的框架，欢迎您尝试LIDE。

    LIDE框架是我们实践使用了至少10年的框架，使用的部分技术或许有点古老，但是，放心，他能支撑上万，上几十万用户量，支持上万并发的框架，他很高效，很强大，很好的扩展，你可以认为这个框架就是一堆的工具集，通过一定的思想组合起来，为你的业务开发铺路用的。
    
    框架包含很多预置的模块，使你可以一上来就拥有基本的系统内容，不必为权限、数据字典、通讯、数据转换、缓存、集群、定时任务...这些基础工作来耗费时间，你的开发就从你的业务编码开始。
